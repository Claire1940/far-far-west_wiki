## Troubleshooting Checklist for Far Far West Crossplay

If friends can’t join, or your team keeps splitting after missions, run this order:

| Problem | Likely Cause | Fast Fix |
|---|---|---|
| **Invite fails instantly** | Privacy mismatch | Set all players to compatible friend/invite permissions |
| **One player can’t queue** | Version mismatch | Restart launcher and force update check |
| **Party disbands between missions** | Leader transfer or session timeout | Recreate party with stable host, then queue immediately |
| **Lag in objective interactions** | Region distance or host instability | Reassign host to lowest average ping player |
| **Desynced boss behavior** | Network spikes under heavy effects | Lower extra modifiers and retest consistency |

> **Tip:** Save one “control run” with no extra modifiers. If that run is stable but chaos runs fail, the issue is likely performance load, not baseline crossplay compatibility.

For content creators or competitive grinders, keep a small tracking sheet with date, build number, host platform, and failure point. In 2026 live-service environments, this is the fastest way to identify patterns.
