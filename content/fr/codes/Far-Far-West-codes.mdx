## Comment échanger correctement les codes Far Far West

Suivez exactement ces étapes pour éviter les échecs de saisie :

1. Ouvrez **Far Far West** et chargez votre menu/hub habituel.
2. Repérez la **zone de saisie promo/code** (elle peut changer après les mises à jour).
3. Collez le code directement (évitez la saisie manuelle si possible).
4. Validez et attendez le texte de confirmation.
5. Vérifiez immédiatement les récompenses de votre compte (surtout le nombre de tickets).

Après l’échange, appliquez les tickets via votre parcours d’amélioration :

1. Allez dans **Weapon Shop** (ou votre panneau d’équipement de héros).
2. Sélectionnez l’arme que vous montez activement.
3. Cliquez sur **Customize**.
4. Utilisez **Free Levels** avec votre pile de tickets.
5. Confirmez chaque niveau jusqu’à atteindre votre palier cible.

| Erreur d’échange | Cause probable | Solution |
|---|---|---|
| **“Invalid code”** | Faute de frappe, mauvaise casse, espaces cachés | Collez le code depuis une source fiable et supprimez les espaces avant/après. |
| **“Code expired”** | Fenêtre de récompense limitée dans le temps terminée | Suivez les codes Far Far West plus récents via les canaux officiels/créateurs actifs. |
| **Aucune récompense ajoutée** | Désynchronisation UI ou synchronisation retardée | Rouvrez l’équipement/la boutique, puis reconnectez-vous une fois avant de réessayer. |
| **Impossible de trouver la boîte de code** | UI déplacée dans un patch | Consultez les notes de patch/posts de mise à jour communautaire et les onglets de menu. |

> **Astuce :** Gardez une petite note texte avec des dates “testé aujourd’hui”. Cela évite de retenter des codes morts.
