## Comment lire correctement l’ordre des symboles

Le plus gros point d’échec dans **far far west area 41** est l’interprétation de l’ordre. Les joueurs entrent souvent les symboles dans l’ordre de découverte (de la première région visitée à la dernière). Ça ne fonctionne généralement pas.

Utilisez plutôt ce processus :

1. Notez les quatre symboles.
2. Notez l’indicateur de points de chaque symbole.
3. Triez les symboles selon la séquence de points (logique gauche → droite).
4. Entrez les symboles sur les cadrans de l’Area 41 dans cet ordre trié.
5. Appuyez une seule fois sur confirmer, puis attendez le retour.

| Mauvaise approche | Bonne approche |
|---|---|
| Entrer les symboles selon le trajet | **Entrer les symboles selon l’ordre des points** |
| Tout mémoriser de tête | **Utiliser des captures pour chaque rocher** |
| Spam confirmer après chaque cadran | **Régler tous les cadrans d’abord, puis confirmer** |
| Supposer que le code d’un ami est identique | **Vérifier vos propres symboles à chaque run** |

Comme la randomisation du code peut varier selon l’instance ou les mises à jour, évitez de vous reposer sur de vieilles captures d’autres sessions. Pour **far far west area 41**, des données fraîches de la session sont plus sûres que des codes copiés.
